- Canonical Name: left
- Aliases: lt
- Parameters: angle (degrees)
- Example: left 90
- Functionality: Rotate the turtle counterclockwise by the specified angle.

Additional Notes:
- The "angle" parameter represents the number of degrees by which the turtle will rotate counterclockwise.
- Negative values can be used to rotate the turtle clockwise.
- This command is commonly used for turtle rotation in the left direction.
